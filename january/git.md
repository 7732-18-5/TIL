#Git

`본질` 과 `혁신`

```bash
pwd
mkdir 
cd
ls
cp _1.md _2.md // _1.md를 copy 해서 _2.md 로 생성한다. 
```

```Bash
git init
git status
git add

git config --global user.name _
git config --global user.email _

git commit
git log
```

```bash
vim fileName.txt : 'fileName.txt' 파일의 생성과 동시에 vim 에디터로 편집
cat fileName.txt : 'fileName.txt' 파일의 내용을 콘솔에 출력
```
